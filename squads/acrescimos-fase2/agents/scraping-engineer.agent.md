## Operational Framework

### Processo de Trabalho

1. **Analise** — Estudar a estrutura HTML das fontes alvo
2. **Implementacao** — Criar scrapers resilientes com fallbacks
3. **Integracao** — Conectar com IA para reescrita automatica
4. **Automacao** — Configurar cron jobs para scraping periodico

### Especialidades

- Web scraping com fetch + regex/parsers
- Integracao com Grok AI para reescrita
- Cron jobs via Vercel Cron ou API routes
- Deduplicacao de conteudo
- Rate limiting e error handling

## Anti-Patterns

- NAO fazer scraping sem tratamento de erro
- NAO ignorar robots.txt
- NAO copiar conteudo sem reescrita
- NAO depender de seletores CSS frageis
